# Init Project

---

使用webpack，eslint，babel构建一个简单的初始化项目，待扩展。。。

### 使用
```
git clone git@github.com:yuelight/Init-project.git proj
cd proj
yarn install
```

### 编译
`npm run build`

### eslint检查
`npm run eslint yourfile`

### 规范化
`npm run lint yourfile`

### editorconfig fix
`npm run indent yourfile`
